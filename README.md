# Madhavi Portfolio Website

A modern, responsive Full Stack Developer Portfolio Website built using HTML, CSS, and JavaScript.

https://your-portfolio-link.com

This portfolio website is designed to showcase my journey as a Full Stack Developer. It includes sections for skills, projects, experience, and a contact form with a focus on performance, responsiveness, and modern UI design.

Features
- Dark / Light Theme Toggle (with local storage)
- Fully Responsive Design (Mobile, Tablet, Desktop)
- Smooth Scroll & Reveal Animations
- Skills Section with categorized tech stack
- Projects Section with detailed cards
- About Section with timeline & experience
- Animated Skill Progress Bars
- Contact Form with validation
- Interactive UI with modern design aesthetics

Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- JavaScript (Vanilla JS)

Concepts Used
- Responsive Web Design
- DOM Manipulation
- Intersection Observer API
- CSS Animations & Transitions
- Local Storage
